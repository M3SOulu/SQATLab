Instructions: Set environment

  • Download Eclipse from this link This eclipse distribution has the Besouro TDD plugin installed. This plugin logs the development activities and recognizes TDD behaviour.
  • Read the exercise description: Sudoku exercise description
  • Go to the Github repository of the exercise: Sudoku exercise project repository
  • Fork your own copy to your account
  • Install Github Desktop (If you have not it)
  • Clone in Desktop your own copy to your account. You should clone the repository on https://github.com/YourGithubaccountName/Sudoku

Instructions: Set up eclipse project and TDD plugin

  • Import the local project repository into Eclipse (In Eclipse File > Import > General > Existing Projects into Workspace)
  • Open Besouro Plugin (In Eclipse Window > Show View > Other > Other > episodeView
  • Open a source file and start Besouro TDD Plugin by clicking play button on plugin window. You should see "recording Sudoku" in the plugin Window.

Instructions: Coding

  • Remember Test-Code-Refacor cycle
  • Each cycle should be as short as possible
  • Try to focus on small tasks
  • Simplest implementation!! Fake it if the implementation is not obvious
  • REFACTOR (Extract variable, method) When refactoring do not change behavior
  • On TDD plugin window observe your development process. You should see "Test-first", "refactoring" labels

Instructions: Updating your repository and submitting your work

  • On Github Desktop: Commit your changes.
  • On Github Desktop: Pull Request to submit your work to the owner of the original repository
