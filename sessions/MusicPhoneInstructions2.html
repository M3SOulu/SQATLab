Instructions

In order to tackle the second part of the MusicPhone project you need to have a working version of the computeDistance method from the first part.

There are two ways to include the implementation of computeDistance method in your code:

  1. Copy the source from http://bit.ly/MusicPhoneTask1 and replace your implementation in the DeviceManager class.

  2. Open the command line inside the MusicPhone directory you cloned last time and type

      git remote add upstream https://github.com/UniOULU-TOL/MusicPhone
    Followed by 
      git fetch upstream
    and finally 
      git merge upstream/master
