Instructions

In order to tackle the remaing parts of the MusicPhone project you need to have a working version of the computeDistance method from the first part and getRecommendations from the second part.

There are two ways to include the implementation of such methods in your code:

  • Copy the source of computeDistance method from http://bit.ly/MusicPhoneTask1 and replace your implementation in the DeviceManager class.

    • In alternative

  • Open the command line inside the MusicPhone directory you cloned last time and type

      git remote add upstream https://github.com/UniOULU-TOL/MusicPhone
    Followed by 
      git fetch upstream
    and finally 
      git merge upstream/master
